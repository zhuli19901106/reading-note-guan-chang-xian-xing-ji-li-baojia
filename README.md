# 李宝嘉《官场现形记》读书笔记

李宝嘉，字伯元，晚清著名小说家。这本《官场现形记》是他最经典的谴责小说。  
全书存本仅六十回，因为作者尚未杀青便因病逝世。  
生死有命，不得不说是一大遗憾。  

相比冯梦龙《三言》比较传统的因果报应+大团圆，《官场现形记》更富有现实意义和指导性。  
因此我选了这本继续阅读，为人处世，时时事事引以为戒。
